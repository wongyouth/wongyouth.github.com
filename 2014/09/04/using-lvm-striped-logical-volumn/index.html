Using LVM striped logical volumn

发表于 2014-09-04 更新于 2025-02-25 分类于 Server Disqus：

之前写过一篇关于 LVM 的博文。
今天要说的是有关 LVM 的一个应用。

原理

当系统需要很多读写操作，需要更高效率的磁盘读写能力，完全升级磁盘等级会没有太多意义。
因为无论多好的磁盘读写能力，总会达到峰顶。

这时候我们需要另一种的解法。

挂载更多的磁盘到一个目录，这样写到一个磁盘的数据会分散到各个磁盘中，
这样子理论上可以达到无限扩展。

能够实现这个技术的，有RAID0, LVM Stripe。这里我只说LVM。
